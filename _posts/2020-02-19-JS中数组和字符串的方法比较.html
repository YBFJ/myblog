JS中数组和字符串的方法比较

vuePress-theme-reco CMJ    2021

JS中数组和字符串的方法比较

CMJ 2020-02-19 JS FE Interview

# JS 中数组和字符串的方法比较

学习自于继海的一篇博文链接 (opens new window) 字符串和数组有很多的相同之处，它们方法众多，而且相似度很高；但是又有不同之处，字符串是不可变的值，可以把它当做只读的数组。 学习这个也通用的理解了一个问题，call 和 apply 方法，他们都是改变函数作用域，第一个参数都是要把 this 修改成的对象，当函数需要参数的时候，call 是使用单个的参数进行传递，而 apply 是使用数组进行参数的传递；当使用 call 和 apply 传入的第一个参数为值的类型的时候，会将值的类型转换成相应的对象(引用类型)，然后复制给 this；并且当传入的第一个参数为 null 或者 undefined 的时候，会把 this 的值赋值给 Windows，并且表示为函数调用模式。

# 是否可索引 String Array

String 和 Array 都是可索引的，用[]代替了 CharAt()进行索引的调用，更加简洁、可读并且更高效。如果参数超出范围或者是 NaN 时，则输出 undefined。

# String 和 Array 进行相互转换

字符串可以使用 split()方法转换成数组；而数组可以使用 join()方法转换成字符串。

# [split()] String

split()基于指定的分隔符将一个字符串分割成多个字符串，并将结果放在一个数组中，分隔符可以是字符串，也可以是一个正则表达式。 split 方法可以接受第二个参数用于指定数组的大小，如果第二个参数为 0-array.length 范围内的值的时候，按照指定参数输出，其他情况将多有的结果都输出。 如果指定的分隔符没有出现在字符串中，则以数组的形式返回原字符串的值。

var colorText = "red,blue,green,yellow";
console.log(colorText.split(/e/)); //["r", "d,blu", ",gr", "", "n,y", "llow"]
console.log(colorText.split(/[^\,]+/)); //将除去逗号以外的字符串变为分隔符["", ",", ",", ",", ""],IE8-会识别为[",",",",","]

# [join()] Array String(Array.prototype.join.call())

join()方法可以使用不同的分隔符来构建这个字符串，join()方法只接受一个参数，也就是用于作为分隔符的字符串，然后返回包含所有数组项的字符串。如果不给 join()方法传入任何值，则使用都还作为分隔符。 如果数组中某一项的值是 null 或者 undefined，则在 join 方法返回的结果中以空字符串表示

var colors = [1, undefined, 2, null, 3];
console.log(colors.join()); //'1,,2,,3'

因为字符串是类数组对象，所以也是可以使用 join()方法。 可以直接通过原型.call 使用但是这里不能用 apply 因为只有'-'一个参数，也可以先 split 之后再 join 使用。

console.log(Array.prototype.join.call("hello", "-")); // "h-e-l-l-o"
var str = "test";
var arr = str.split(""); //["t", "e", "s", "t"]
console.log(arr.join("-")); //'t-e-s-t'

# 拼接

# [concat()] String Array

字符串和数组共同拥有拼接方法 concat()。 总而言之，concat 方法可以随意使用，可以数组拼接字符串，则拼接到一个数组里面；也可以数组拼接数组，还是拼接到一个数组里面；还可以数组拼接数组中的数组。 字符串可以拼接字符串，变成连在一起的一个字符串；字符串可以拼接一个数组，也还是变成一个字符串；还可以拼接数组里面的数组，拼出来还是一个字符串。 这个拼接带点拼接成同样类型的意味儿。

var value = "hello";
console.log(value.concat("world")); //'helloworld'
console.log(value.concat(["world"])); //'helloworld'
console.log(value.concat([["world"]])); //'helloworld'
var value = ["hello"];
console.log(value.concat("world")); //["hello", "world"]
console.log(value.concat(["world"])); //["hello", "world"]
console.log(value.concat([["world"]])); //["hello", ["world"]]

# 创建

# [slice()] String Array

字符串和数组都拥有创建方法 slice()，分别用于创建字符串和子数组。 slice()方法基于当前数组或者字符串中的一个或多个项创建一个新的数组或者字符串，接受一个或者两个参数，级要返回项目的起始和结束位置，最后返回新的数组或者新的字符串。 slice(start,end)方法需要两个参数 start 和 end，返回这个数组或者字符串中从 start 位置到（不包含）end 位置的一个子数组或者子字符串；如果 end 为 undefined 或者不存在，则返回从 start 位置到数组（或者字符串）结尾的所有项。

如果 start 为负数，则 start = max(length + start,0) 如果 end 是负数，则 end = max(length+end,0) 其中 start 和 end 无法交换位置

var stringValue = "hello world";
console.log(stringValue.slice(-2, 20)); //'ld'

# 位置

# [indexOf()] && [lastindexOf()] String Array

字符串和数组都拥有查找位置的两个方法，位置方法和中括号[]索引读取方法刚好相反，前者是通过项查找索引，[]则是通过索引查找项。更像是判断某项是否存在于字符串或者数组中一样。

# [indexOf()] String Array

indexOf(search,start)方法会接受 search 和 start 两个参数，返回 search 首次出现的位置，如果没有找到就返回-1 字符串中的 search 参数会调用 String()转型函数，将该参数的非字符串转换成字符串；而数组中的 search 参数则使用严格相等运算符(===)进行比较。不论是数组还是字符串，第二个参数 start 都会隐式调用 Number()转型函数，将 start 非数字值(undefined 除外)转换成数值；若忽略该参数或者该参数为 undefined、NaN 时，start = 0.（数组和字符串都可以正常查找） 若 start 参数为负数，字符串的处理是将 start=0；而数组的处理是 start = max(0,start+length)。

# [lastindexOf()] String Array

与 indexOf 相反，lastindexOf 方法是从右向左寻找，其中大部分是相同的，但是如果胡烈 start 参数或者该参数为 undefined、NaN 是，字符串的处理方式是 start = length -1；而数组的处理是 start =0. （上面一段话简而言之，last 反向查找的时候，start 为 undefined NaN 时，如果数组中查找的数字不是在 0 位置，则返回-1 表示找不到，字符串可以正常查找） 如果 start 参数为负数，字符串的处理是将 start = 0；而数组的处理是 start = max(0,start+length)。

var string = "hello world world";
console.log(string.lastIndexOf("ld", undefined)); // 15

var arr = ["a", "b", "c", "d", "e", "a", "b"];
console.log(arr.lastIndexOf("c", undefined)); // -1

console.log(arr.lastIndexOf("b", undefined)); // -1

# 顺序

# [reverse()]  Array String(Arry.prototype.reverse.apply())

在数组 Array 中存在 reverse 来反转数组，但是 String 中也可以使用 call()或者 apply()来实现反转 由于 reverse 不能传参，所以用 apply 方法将 split 完成的字符串传进去，再用 join 转成字符串

var str = "12435";
console.log(str, Array.prototype.reverse.apply(str.split("")).join(""));

# [sort()]  Array String(Array.prototype.sort.apply())

在数组 Array 中存在 sort()方法对数组排序，默认按字符串升序排列数组项

var array = [1, 2, 4, 3, 5];
console.log(array, array.sort());

而字符串也可以使用 call 或者 apply 来实现排序

var str = "12435";
Console.log(str, Array.prototype.sort.apply(str.split("")).join(""));
